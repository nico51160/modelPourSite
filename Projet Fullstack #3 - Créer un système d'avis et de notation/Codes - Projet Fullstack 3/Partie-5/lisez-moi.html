Projet fullstack: Avis et notation

Dans cette 5ème partie, nous allons mettre en place l'objet XMLHttpResquest de JavaScript afin de produire une animation pour une meilleure expérience utilisateur.
L'animation sera de faire disparaître la fenêtre contenant le formulaire de vote, puis de faire apparaître la réponse json envoyée par le Php.

Connaissances requises :

  • JavaScript moderne avancé - AJAX. XMLHttpRequest
  • Comment ajouter de l'animation à votre site facilement
