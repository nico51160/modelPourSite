Projet fullstack: Gestion d'une commande (Partie Admin & Partie Public)

Dans cette 5ème partie, nous allons afficher la référence de chaque commande dans un menu déroulant dont la zone apparaitra ou disparaitra par simple clic.
Nous mettrons en place un loader afin de faire patienter l'utilisateur pendant que le retour des références s'affichent dans le menu déroulant.

Connaissances requises :

  • Javascript Moderne : Le bundle
  • Les différents types de positionnement en CSS
  • Créer un loader circulaire en CSS
