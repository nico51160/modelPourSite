Projet fullstack: Gestion d'une commande (Partie Admin & Partie Public)

Dans cette 4ème partie, nous allons créer la BDD, instancier la classe PDO, créer la classe d'entité Commande ainsi que la classe manager CommandeManager.
Le but de cette partie sera d'afficher la référence de chaque commande se trouvant dans la table Commande.

Connaissances requises :

  • Tout savoir sur la Classe PDO
  • La POO en PHP : L'héritage
  • JavaScript moderne avancé - AJAX. XMLHttpRequest
  • Programmation Orientée Objet en PHP
  • API Rest de A à Z
