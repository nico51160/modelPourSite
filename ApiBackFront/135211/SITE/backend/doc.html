Documentation

  • Pour afficher tous les tutos :
    http://127.0.0.1:8080/API_REST/backend/tutos/api/readAll
    Méthode: GET
  • Pour afficher un tuto :
    http://127.0.0.1:8080/API_REST/backend/tutos/api/read/$1
    Méthode: GET
  • Pour créer un tuto :
    http://127.0.0.1:8080/API_REST/backend/tutos/api/create
    Méthode: POST
  • Pour modifier un tuto :
    http://127.0.0.1:8080/API_REST/backend/tutos/api/update
    Méthode: PUT
  • Pour supprimer un tuto :
    http://127.0.0.1:8080/API_REST/backend/tutos/api/delete
    Méthode: DELETE
